### Local Installation :

> Clone the repository:

```sh
git clone https://github.com/PrajjwalDatir/Weatherly.git
cd Weatherly
```
> Install the dependencies:
```sh
npm ci
```
> Running the webapp:
```sh
npm start
```
> Building the webapp:
```sh
npm build
```
